Utilizar biblioteca de terceiros

  1. Verifique se o tsd está instalado (tsd --version) - NPM site
  2. Inicie o gerenciar de pacote do tsc (tsd init)
  3. Para pesquisar: tsd query knockout --info --resolve
  4. Para instalar: tsd install knockout --resolve --save
  5. Se preferir pode ser via nuget: procure por knockout.typescript
